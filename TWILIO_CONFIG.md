# Project X - Twilio Configuration

## Credentials
- **Account SID:** Use TWILIO_ACCOUNT_SID environment variable
- **Auth Token:** Use TWILIO_AUTH_TOKEN environment variable
- **Service:** AI Phone Receptionist

## How It Works
1. Buy Twilio phone number (~$1/month)
2. Configure webhook to OpenAI GPT-4
3. AI answers calls, books appointments, qualifies leads
4. Sends summary to client via SMS/email

## Cost per Client
- Phone number: $1/month
- Calls: ~$0.02/minute
- Average 100 min/month = $2
- **Total cost:** ~$3/month per client
- **Charge client:** $500/month
- **Margin:** $497/month per client
